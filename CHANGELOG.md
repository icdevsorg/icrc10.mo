# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-02-24

### Added

- Mixin pattern (`mo:icrc10-mo/mixin`) for automatic endpoint generation using `persistent actor class` and Class+

### Changed

- Migrated to `mo:core` (replacing `mo:base`, `mo:map`, `mo:vector`)
- Migrated to Motoko 1.1.0 with Enhanced Orthogonal Persistence
